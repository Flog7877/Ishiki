Eine Mailboxanfrage stellen

Manche Therapeut*innen bieten keine direkten telefonischen Sprechzeiten. Doch das ist kein Problem, denn das ermöglicht eine Anfrage zu einem beliebigen Zeitpunkt und Flexibiltät für beide Seiten - unabhängig von nervigen Telefonzeiten.

Die Anfrage

Um eine Anfrage per Mailbox zu stellen, muss natürlich die Nummer gewählt werden. Nach einer kurzen Zeit geht dann der Anrufsbeantworter ran. In manchen Fällen wird zuvor ein Band abgespielt, was für Informationen die Therapeut*innen benötigen. Es reicht ein kurzer Satz, dass man einen Therapieplatz suche und sich über einen Rückruf freuen würde. Im Allgemeinen sollten dann noch folgende Infos hinterlassen werden:
  • Voller Name
  • Telefonnummer
  • Erreichbarkeitszeiten
  • Versicherungsstatus (Privat/ Gesetzlich)
Falls man sich dazu bereit fühlt, können Symptome oder der Grund, weshalb eine Therapie gesucht wird, kurz angeschnitten werden. Das ist aber keine Voraussetzung! Das Ziel der Mailboxanfrage ist eine erste Kontaktaufnahme.
